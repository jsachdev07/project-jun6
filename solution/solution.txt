Steps --> 

1. Install docker 
	apt install docker.io -y 

2. Jenkins user run pipeline and jenkins user should have permission to run docker commands 

Any one method should be used 

	- give Ownership to jenkins group on docker socket file --> chown root:jenkins /var/run/docker.sock
	- add jenkins user in docker group --> usermod -a -G docker jenkins 
	
3. write dockerfile and push it on github

dockerfile --> 

FROM iamdevopstrainer/tomcat:base
COPY abc_tech.war /usr/local/tomcat/webapps/
CMD ["catalina.sh", "run"]

4. the image created by executing dockerfile is present locally on the jenkins server I need to push it to dockerhub
   docker push command will be executed as part of my pipeline but for docker push to be successful we need the credentials of dockerhub account to be saved on jenkins global credentials.

you need to make sure your dockerhub username and password is stored in jenkins global credentials with the credential id as "dockerhub"


5. how will you login in your pipeline workflow? 

that's where you need to install a docker pipeline plugin in jenkins and restart jenkins 

withDockerRegistry() --> method will be used to login on dockerhub and this method is provided by the docker pipeline plugin that we have to install as a pre-requisite.

6. create a ci-cd job on jenkins and deploy app in a container 

jobname --> ci-cd-job


pipeline
{
	agent any
	stages
	{
		stage('Checkout')
		{
			steps
			{
				git 'https://github.com/jsachdev07/project-jun6.git'
			}
		}
		
		stage('Compile')
		{
			steps
			{
				sh 'mvn compile'
			}
		}

		stage('Test')
		{
			steps
			{
				sh 'mvn test'
			}
		}

		stage('Build')
		{
			steps
			{
				sh 'mvn package'
			}
		}
		
        stage('Build Docker Image') {
            steps {
                sh '''
                    cp target/ABCtechnologies-1.0.war abc_tech.war
                    docker build -t abc_tech:$BUILD_NUMBER .
                    docker tag abc_tech:$BUILD_NUMBER jsachdev07/abc_tech:$BUILD_NUMBER
                '''
            }
        }

        stage('Push Docker Image') {
            steps {
                withDockerRegistry([credentialsId: "dockerhub", url: ""]) {
                    sh 'docker push jsachdev07/abc_tech:$BUILD_NUMBER'
                }
            }
        }
		
		stage('Deploy as container')
		{
			steps
			{
				sh 'docker run -d -P jsachdev07/abc_tech:$BUILD_NUMBER'
			}
		}  
		
	}
}


Artifact --> /var/lib/jenkins/workspaces/ci-cd-job/target/ABCtechnologies-1.0.war
